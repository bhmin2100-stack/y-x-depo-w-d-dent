컨포멀 증착 덴트 모델 - Windows 프로그램

실행 방법
1. dist\DentDepositionModel.exe 를 더블클릭합니다.
2. exe가 없으면 run_windows_app.bat 을 더블클릭합니다.

기능
- 브라우저나 localhost 없이 Windows 창으로 실행됩니다.
- W, D, 최대 적층량, 현재 적층량을 조절합니다.
- 왼쪽 위는 y(x), z(x) 그래프입니다.
- 오른쪽 위는 1 단위 적층 모습을 나이테처럼 보여줍니다.
- 아래는 기본 덴트 구조 점 편집창입니다.

점 편집
- 점을 드래그하면 이동합니다.
- 점 추가 버튼을 누른 뒤 빈 위치를 클릭하면 점이 추가됩니다.
- 점 삭제 버튼을 누른 뒤 점을 클릭하면 삭제됩니다.
- 양 끝 점은 덴트 폭 W의 경계라 삭제되지 않습니다.
