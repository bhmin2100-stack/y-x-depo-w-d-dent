# 컨포멀 증착 덴트 모델

컨포멀 증착으로 덴트 깊이가 얼마나 개선되는지 확인하는 Python/웹 도구입니다.

## 표시 항목

- `x`: 컨포멀 증착량
- `y`: 증착 후 남은 덴트 깊이
- `W`: 덴트 폭
- `D`: 초기 덴트 깊이
- `z`: 평평한 필드(field) 면을 `0도`로 봤을 때 증착 후 상단 표면의 최대 접선각

Streamlit 앱에서는 `y(x), z(x)` 그래프와 `증착 적층` 화면을 확인합니다.

조작은 단순하게 `최대 적층량`과 `현재 적층량` 슬라이더만 사용합니다. 적층층은 1 단위 간격으로 자동 표시됩니다.

`기본 구조 편집` 영역은 PowerPoint 점 편집처럼 덴트의 기본 구조만 보여주는 편집창입니다.

- 덴트 곡선의 흰 점을 드래그해서 기본 구조 수정
- 편집 영역을 더블클릭해서 새 점 추가
- 내부 점을 선택한 뒤 `점 삭제` 버튼이나 Delete 키로 삭제
- 기본 구조 편집창은 화면 제일 아래에 배치

그래프와 쌓이는 모습은 위쪽에서 양옆으로 표시됩니다.

- 왼쪽: `y(x)`, `z(x)` 그래프
- 오른쪽: 필드 평탄부까지 포함한 나이테식 적층 모습
- 각 그래프 축에는 격자 숫자를 표시

## 실행

### iPad/브라우저에서 기본 구조 편집

GitHub Pages 주소:

```text
https://bhmin2100-stack.github.io/y-x-depo-w-d-dent/
```

이 정적 웹 버전은 `index.html` 하나로 동작하며, Python 설치 없이 iPad Safari/Chrome에서 기본 덴트 구조를 점 편집할 수 있습니다.

### PC에서 Streamlit으로 실행

```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

또는 아래 파일을 더블클릭합니다.

```text
run_app.bat
```

## 모델

앱은 2D 덴트 경계를 원래 노출 표면으로 보고, 그 경계를 증착 거리 `x`만큼 확장한 형태로 컨포멀 증착을 계산합니다. 증착된 필드 상단 높이는 `x`입니다. 남은 덴트 깊이는 다음처럼 계산합니다.

```text
y = 증착 후 필드 상단 높이 - 덴트 내부의 가장 낮은 증착 표면 높이
```

각도 `z`는 증착 후 덴트 내부 상단 경계의 최대 절대 기울기에서 계산합니다.

```text
z = atan(max(abs(dy/dx)))
```
